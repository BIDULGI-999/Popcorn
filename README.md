# PopCorn

팝업 스토어 통합 예약 및 개인화 추천 플랫폼

---

## 프로젝트 소개

**PopCorn**은 여러 플랫폼에 분산된 팝업 스토어 정보를 한 곳에서 통합 관리하고,
대규모 트래픽 환경에서도 안정적인 예약과 사용자 맞춤형 추천을 제공하는
팝업 스토어 통합 예약 플랫폼입니다.

인기 팝업 스토어의 예약 오픈 시 발생하는 트래픽 집중 문제를 해결하고,
사용자의 관심사와 활동 지역을 기반으로 효율적인 팝업 탐색 경험을 제공하는 것을 목표로 합니다.

---

## 기획 배경

최근 한정성과 경험 중심 소비 트렌드 확산으로 팝업 스토어에 대한 수요가 급격히 증가하고 있습니다.
그러나 현재 팝업 스토어 서비스는 다음과 같은 한계를 가지고 있습니다.

* 플랫폼별로 분산된 예약 시스템
* 인기 팝업 예약 시 트래픽 폭주로 인한 서버 불안정
* 사용자 취향을 고려하지 않은 단순 정보 제공

PopCorn은 이러한 문제를 해결하기 위해
**통합 예약**, **대기열 기반 트래픽 제어**, **개인화 추천 시스템**을 중심으로 설계되었습니다.

---

## 핵심 기능

### 통합 예약

* 여러 팝업 스토어 예약 정보를 하나의 플랫폼에서 조회 및 예약
* QR 기반 예약 확인 및 입장 처리 지원

### 대기열 기반 트래픽 제어

* 접속 순서 기반 가상 대기열 생성
* 서버 가용량만큼만 순차적으로 요청을 처리하여 트래픽 폭주 방지
* 실시간 대기 순번 확인 가능

### 개인화 추천

* 사용자 선호 카테고리 및 활동 지역 데이터 기반 추천
* Redis 캐싱을 활용한 빠른 응답 속도 제공

---

## 차별화 요소

기존 팝업 예약 서비스가 단순 예약 기능에 집중했다면,
PopCorn은 대규모 트래픽 상황에서도 안정적으로 동작하는 시스템 구조와
사용자 맞춤형 추천 경험 제공에 초점을 맞췄습니다.

* Kafka 기반 비동기 이벤트 처리로 트래픽 분산
* Redis를 활용한 대기열 관리 및 추천 데이터 캐싱
* 위치 + 카테고리 기반 큐레이션 로직 적용
* MSA 기반 구조로 서비스 확장성과 유지보수성 확보

---

## 시스템 아키텍처

* MSA(Microservice Architecture) 기반 설계
* 각 도메인(User, Reservation, Product, Payment, Recommendation)을 독립 서비스로 분리
* Kafka를 활용한 이벤트 기반 통신으로 서비스 간 결합도 최소화
* Redis 캐싱 전략을 통해 조회 성능 최적화

---

## 주요 구현 내용

### 대기열 관리 (Traffic Control)

* 대량의 동시 접속 요청 발생 시 서버에 직접 처리하지 않고 대기열에 적재
* 가용 처리량만큼만 요청을 순차적으로 허용하여 시스템 안정성 확보

### 이벤트 기반 예약 처리

* 예약 요청 시 DB에 즉시 쓰기 대신 Kafka Topic으로 이벤트 발행
* Consumer에서 예약 이벤트를 처리하여 데이터 영속화
* 트래픽 폭주 상황에서도 메시지 유실 없이 안정적인 처리 보장

### 상품 예약 동시성 제어

* 분산 락을 활용한 실시간 경쟁 상황 처리
* 중복 예약 및 데이터 정합성 문제 방지

### 결제 시스템

* 트랜잭션 기반 결제 흐름 설계
* 결제 실패 및 중복 처리 방지 로직 구현

### 추천 시스템 성능 최적화

* Redis 스케줄러를 활용해 유저별 추천 데이터를 주기적으로 선계산
* 요청 시 복잡한 DB Join 없이 Redis에서 즉시 응답

## 기술 스택

### Backend

* Java 17
* Spring Boot
* Spring Data JPA

### Messaging

* Apache Kafka

  * 예약 요청 및 대규모 트래픽 비동기 처리

### Cache / In-Memory DB

* Redis

  * 대기열 관리
  * 추천 결과 캐싱
  * 스케줄러 기반 선연산 처리

### 기타

* QR Code 라이브러리
* 실시간 알림 시스템 (FCM 또는 Webhook)

## 학습 및 훈련 내용과의 연관성

* MSA 기반 서비스 설계를 통한 독립 배포 및 유지보수 경험
* Kafka를 활용한 이벤트 기반 아키텍처 설계 및 구현
* 분산 시스템 환경에서의 동시성 제어 및 트래픽 관리 경험
